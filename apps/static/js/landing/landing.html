How is this site built?

This site is built using Django, an awesome Python web framework, AngularJS, A Front End Javascript framework, and Twitter Bootstrap, an easy way to make sites friendly and pretty. For building the API I used Django Rest Framework (DRF), which makes it easy to implement and API over an existing database and models.

There is even an android client thanks to the API! Go check it out \o/
